# Herramientas para APIs y monitoreo

## Axios
Axios es una librería JavaScript para hacer peticiones HTTP desde el navegador o Node.js. Permite realizar solicitudes asíncronas a APIs REST de forma sencilla y soporta promesas, interceptores y cancelación de peticiones.

**¿Por qué se usa?**
- Sintaxis más simple y potente que fetch.
- Mejor manejo de errores y respuestas JSON.
- Soporta peticiones y respuestas interceptadas (útil para autenticación, logs, etc).

## Postman
Postman es una herramienta gráfica para probar, documentar y automatizar APIs REST. Permite enviar peticiones HTTP, ver respuestas, guardar colecciones y generar documentación.

**¿Por qué se usa?**
- Facilita el testeo manual de endpoints.
- Permite automatizar pruebas y compartir colecciones.
- Útil para depuración y documentación de APIs.

## Sentry
Sentry es una plataforma de monitoreo de errores y rendimiento para aplicaciones web y móviles. Captura excepciones, errores y trazas de rendimiento en tiempo real.

**¿Por qué se usa?**
- Detecta errores en producción automáticamente.
- Proporciona alertas y reportes detallados.
- Ayuda a mejorar la calidad y estabilidad del software.

## Swagger
Swagger (OpenAPI) es un estándar y conjunto de herramientas para documentar, diseñar y probar APIs REST. Permite describir endpoints, parámetros y respuestas en un formato legible por humanos y máquinas.

**¿Por qué se usa?**
- Genera documentación interactiva de la API.
- Facilita la validación y el diseño de contratos de API.
- Permite generar clientes y servidores automáticamente a partir del esquema.
